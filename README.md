# flashgod

Ultra-light-weight flash-loan aggregator on all EVM compatible chains.

### Sauce
Unbridled access to __all__ flashloanable assets of __all__ flash-loan providers on __any__ given chain, *__in 1 transaction__*.

Featuring providers:
- Aave V3
- Uniswap V3
- Balancer V2

On chains:
- Ethereum
- Arbitrum
- Optimism
- Polygon
- Base
